FBLA Computer Game and Simulation Programming Abington Senior High School : Ranwei Hu, Arie Dash 
Anachronistic Odyssey version 1.0 12/5/2015
===================================================================================================
===================================================================================================
Welcome to Anachronistic Odyssey, an educational space simulation game. This game was made using the Unity 5 engine and the Javascript scripting language. 
Travel through our universe and solve educational problems to save the world!
===================================================================================================
===================================================================================================
Usage:

To run the game double click on the AnachronisticOdyssey shortcut file

Note: If this does not work, double click on the AnachronisticOdyssey application file.
